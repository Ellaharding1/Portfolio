Java Program: Algorithms

In this project, I developed a Java program to implement and compare various sorting algorithms, including Selection Sort, Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, and Bucket Sort. The program measures and outputs the execution time for each algorithm, allowing for a detailed performance analysis across different sorting methods. This project highlights my understanding of algorithmic efficiency and my ability to apply Java programming to solve computational problems effectively.
