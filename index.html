European Travel Destinations Explorer

This project is a web application designed to explore travel destinations across Europe. Users can browse and filter destinations based on various criteria, such as region, country, language, and more. The application supports list creation, sorting, and deletion, allowing users to organize destinations into personalized collections. A built-in map powered by Leaflet.js displays geographical coordinates for each destination, providing an interactive view of travel spots. The backend is built with Node.js and Express, with data storage in JSON and CSV formats, offering robust API endpoints to retrieve and manage destination information.
